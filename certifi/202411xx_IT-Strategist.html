      # 試験本番

      試験の前日に詰め込むのをやめましょう。普段より早い時間に寝てパフォーマンスを発揮することが大事です。また必要な持ち物は前日に準備しておきましょう。

      ▼わたしの持ち物
      [ ] シャープペン x2本
      [ ] 消しゴム x2個
      [ ] 色付きフリクション x1
      [ ] ゴミ袋 x1袋
      - ゴミは持ち帰りなので
      [ ] 500ml ペットボトルの飲み物 x1
      [ ] ラムネ菓子 x1袋
      - 試験の合間に糖分補給のため
      [ ] その他、最低限必要なもの
      - スマートフォン、財布、家の鍵、Suica..

      # 午前1、午前2に挑む

      午前1、午前2はほとんどが過去問と同じなので、しっかり勉強していればサクサク解けます。問題を後ろから解くことで、残り時間と残り問題数を把握できます(いま問10を解いていて残り30分だから、1問に3分掛けられるな..といった具合)

      # 午後1に挑む

      別の試験ではメモを用意するために問題用紙の最終ページをビリビリ破るという技があります。でもIPAの試験では、問題用紙を破ることは許されていません。注意しましょう。

      # 午後2に挑む

      まず、予定通り各章のタイトルを抽出します。次に、問題本文の記述を元に論述が求められている要素を並べます。
      要素を抽出して書き起こすのに結構な時間が掛かるので、だんだんと焦る気持ちになってきます。整理が終わったのは20分経過した頃ですが、第3問についてはぼんやりとしか書けていない状態で焦る気持ちが高まります。書き始めたら良いストーリーを思いつくだろう、等と言い聞かせて本文を書き始めます。
